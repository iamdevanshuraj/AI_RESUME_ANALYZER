# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, and Groq API.

## Features

- Upload Resume PDF
- Paste Job Description
- ATS Score Analysis
- Missing Skills Detection
- Strength Analysis
- Improvement Suggestions

## Tech Stack

- Python
- Streamlit
- Groq API
- PyPDF2

## Installation

pip install streamlit groq pypdf2

streamlit run app.py
